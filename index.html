Visão Geral

Este guia explica como instalar, usar e solucionar problemas do aplicativo — com imagens de tela, passos e dicas.

Modo de exibição:

Instalação

Instruções gerais para instalar o aplicativo em Android / iOS / Desktop (PWA).

Android (APK)

  1. Baixe o arquivo .apk pelo link fornecido pelo time.
  2. Permita instalações de fontes desconhecidas nas configurações do dispositivo.
  3. Abra o APK e siga o instalador.

iOS / TestFlight

  1. Receba o convite do TestFlight.
  2. Instale via TestFlight e aceite as permissões solicitadas.

Desktop (PWA)

Abra no navegador e instale como PWA (opção "Instalar" no menu do navegador).

Passo a Passo — Primeira execução

  1. Abra o aplicativo e crie uma conta (ou faça login).
  2. Siga o assistente inicial para configurar preferências básicas.
  3. Conceda permissões solicitadas (localização, notificações) se necessário.
  4. Crie seu primeiro item usando o botão "+" no Dashboard.
  5. Verifique se os dados aparecem em relatórios e, se desejar, faça um backup.

Exemplo de como criar um item

// Pseudocódigo: criar item
app.open('new')
form.title = 'Meu item'
form.save()
Sincronização manual

Vá em Configurações → Backup → Sincronizar agora.

Dicas & Boas Práticas

  • Mantenha o app atualizado para corrigir bugs e ter novas funcionalidades.
  • Ative backups automáticos para evitar perda de dados.
  • Use conexões seguras (Wi‑Fi confiável) ao sincronizar grandes volumes.
  • Se tiver dados sensíveis, ative bloqueio por PIN/biometria.

FAQ

O app funciona offline?

Sim — é possível usar offline. As ações ficarão em fila e serão sincronizadas quando o dispositivo reconectar à internet.

Como recuperar a senha?

Na tela de login clique em "Esqueci a senha" e siga as instruções enviadas por e-mail.

Preciso pagar para usar?

Existem planos gratuitos e pagos. Consulte a tela de planos dentro do app.
